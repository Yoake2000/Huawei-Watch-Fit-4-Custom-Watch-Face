# Huawei-Watch-Fit-4-Custom-Watch-Face
A repository of custom watch faces I made/imported for the Huawei Watch Fit 4
- Custom watch faces created using [Facemaker](https://facemaker.pt/)
- Watch face project folders and hwt files are uploaded

# List of Watch Faces
## Suisei Watch
- Imported from a Clockology watch face for Apple Watches. Original watch face made by [@modeee0709](https://x.com/modeee0709/status/1506784449018875905). Image files used in importing are extracted from the watch face using the clockology ios app.
